# Frontend (React + Vite)

Como rodar:
1. Entrar na pasta `frontend`:
   - Powershell: `cd frontend`
2. Instalar dependências:
   - `npm install`
3. Rodar em modo dev:
   - `npm run dev`
   - A aplicação estará em `http://localhost:3000`
4. Certifique-se que a API .NET está rodando e que `Program.cs` permite CORS para `http://localhost:3000`.

Observação:
- Ajuste `VITE_API_URL` em `.env` se sua API rodar em outra porta.
